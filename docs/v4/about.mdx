# 关于

## 声明

:::warning{title=使用声明}
文档 v3 版本内容基于 SailWind 3.0 内容构建，v4 版本内容基于 SailWind 4.0 内容构建。

目前主要维护的版本为 **`v4`**。

仅限 **`派兹互联-测试部`** 内部使用，不具备官方支持。
:::

## 文档更新策略

- 本文档通过外网仓库 <SourceCode href="https://github.com/mikigo/sailwind_docs" /> 维护更新；
- 内网文档仓库，会定期从外网仓库做内容同步；
- 欢迎任意的内容贡献，请以 Pull Request 的方式提交到外网仓库；

有任何问题请联系：huangmingqiang@pzeda.com；
